# 项目名称

## 项目简介
该项目是一个基于 Vue.js 的单页面应用，旨在提供一个简单的用户界面来管理个人资料、目标待办事项和通知公告。用户可以通过导航菜单在不同的页面之间切换。

## 功能特性
- **个人资料展示**：在主页上显示用户的姓名和学号。
- **目标管理**：用户可以在目标页面添加、删除和提交目标。
- **通知公告**：用户可以查看通知列表，并通过分页功能浏览不同的通知。
- **组件切换**：通过侧边栏菜单，用户可以在主页、通知页面和目标页面之间切换。
